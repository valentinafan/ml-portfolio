# Portfolio de Data Science & Machine Learning

Coleccion de proyectos de analisis de datos y machine learning desarrollados con Python. Cada proyecto incluye exploracion de datos, visualizaciones profesionales y conclusiones basadas en evidencia.

## Proyectos

| # | Proyecto | Descripcion | Tecnicas |
|---|----------|-------------|----------|
| 01 | [Costos de Seguros Medicos](01-medical-insurance-costs/) | Regresion lineal multiple desde cero (OLS) para predecir costos de seguros | Regresion OLS, algebra matricial, inferencia estadistica |
| 02 | [Prediccion de Obesidad](02-obesity-feature-selection/) | Comparacion de 3 metodos wrapper para seleccion de features | SFS, SBS, RFE, regresion logistica |
| 03 | [Esperanza de Vida y PIB](03-life-expectancy-gdp/) | Estudio comparativo entre 6 paises (2000-2015) | EDA, correlacion, regresion, t-tests |
| 04 | [Biodiversidad en Parques Nacionales](04-biodiversity-national-parks/) | Analisis de conservacion de 5,824 especies en 4 parques | Chi-cuadrado, EDA, preparacion ML |
| 05 | [Hongos: Comestibles vs Venenosos](05-exploring-mushrooms/) | Identificacion de patrones de toxicidad en 8,124 hongos | V de Cramer, analisis categorico |
| 06 | [Rendimiento en Tenis Profesional](06-tennis-ace-performance/) | Modelos predictivos de ganancias en tenis | Regresion simple y multiple, sklearn |
| 07 | [Exploracion de Jeopardy](07-jeopardy-data-exploration/) | Analisis de 216,000+ preguntas del programa de TV | Big data, NLP basico, analisis temporal |

## Stack Tecnologico

- **Python 3.x** - Lenguaje principal
- **Pandas & NumPy** - Manipulacion y analisis de datos
- **Matplotlib & Seaborn** - Visualizaciones
- **Scikit-learn** - Machine Learning
- **SciPy** - Estadistica y pruebas de hipotesis
